# Skill Template

Copy this entire folder to start a new La-Z-Boy skill.

## Usage

```bash
cp -r assets/skill-template/ ../your-new-skill-name/
```

Then:
1. Rename the folder to your skill name (kebab-case)
2. Edit `SKILL.md` — fill in every `[placeholder]`
3. Add content to `references/`, `scripts/`, `assets/` as needed
4. Delete any folders you don't use (don't commit empty folders)
5. Run through the quality checklist in `lazboy-skill-standard/SKILL.md`
6. Register your skill in `lazboy-skill-standard/references/skill-registry.md`
